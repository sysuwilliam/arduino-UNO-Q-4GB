## 概述

UNO Q 是一款功能强大、多用途的开发板，专为爱好者、开发者和创客设计。得益于其混合架构，我们可以将其作为功能完整的基于 Linux 的**单板计算机（SBC）**来使用，同时能够运行 Arduino sketch。

![UNO Q 单板计算机](https://docs.arduino.cc/static/943dfd18c0823bb1de07ed1b61dca087/a6d36/thumbnail.png)

在本教程中，您将学习如何将 UNO Q 设置为个人计算机，可用于网页浏览、办公和媒体播放等日常任务，同时还可以使用 **Arduino App Lab** 开发精彩项目。

- 探索独立 Arduino App Lab 模式

## 所需硬件和软件

要将 UNO Q 设置为单板计算机，您需要以下硬件和软件组件。

![UNO Q SBC 硬件](https://docs.arduino.cc/static/ecb03c920bb1270b989faaea8badfb28/a6d36/hardware.png)

### 硬件要求

- [Arduino® UNO Q](https://store.arduino.cc/products/uno-q)
- USB 鼠标
- USB 键盘
- HDMI 显示器
- USB-C® 转接器（Dongle）¹
- 电源适配器 +5 VDC 3 A（为 USB-C 转接器供电）

> ⚠️ **注意：** Apple 的 USB-C 转接器已经测试与 UNO Q 不兼容，请选择其他替代型号。

### 软件要求

UNO Q 预装了基于 Debian 发行版的桌面操作系统（具有上游支持），让您可以立即开始工作。对于本教程中的测试，无需刷写开发板或安装任何额外的镜像或程序。

---

## 单板计算机用途的 UNO Q 特性

UNO Q 提供日常计算和嵌入式应用所需的性能和连接能力。

- **CPU：** Qualcomm Dragonwing™ QRB2210 | 四核 Arm® Cortex®-A53 @ 2.0 GHz
- **GPU：** Adreno 702 GPU @ 845 MHz | 3D 图形加速器
- **RAM：** 2 GB 或 4 GB LPDDR4（推荐 **4 GB** 版本以获得流畅的 SBC 体验）
- **存储：** 16 GB 或 32 GB eMMC
- **连接：** Wi-Fi® 5 双频 2.4/5 GHz 和 Bluetooth® 5.1
- **显示：** 通过 USB-C 输出 HD 分辨率

## 计算机组装

按照以下步骤组装您的 UNO Q 单板计算机：

![组装过程](https://docs.arduino.cc/e265826ec2406bdf7ee36be153ff26bc/assembly.gif)

1. 将 USB-C 转接器连接到 Arduino UNO Q 的 USB-C 端口。
2. 将 USB 键盘和鼠标插入转接器的 USB-A 端口。
3. 将显示器连接到转接器的 HDMI 端口。
4. 将外部电源适配器连接到转接器，但暂时不要通电。

> ⚠️ **注意：** UNO Q 不会向 USB-C 转接器供电。转接器必须外部供电，反过来它也会为开发板供电。

### 供电选项

要为您的单板计算机供电，使用外部电源适配器为 USB-C 转接器供电：

![系统供电](https://docs.arduino.cc/static/ed561af338dfe327f0d57231baae9a14/a6d36/power.png)

> ⚠️ **注意：** 电源适配器应至少提供 +5 VDC 3 A，以可靠地为转接器、连接的设备和 UNO Q 供电。

通电后，单板计算机将自动开启并启动。

## 登录和首次使用

当 UNO Q 完成启动后，Debian 登录界面将出现。默认用户名是 `arduino`，在密码字段输入一个**自定义密码**：

- **用户名：** arduino
- **密码：** `new-password`
- 然后您将被要求确认密码。

![登录界面](https://docs.arduino.cc/static/2942e462365d34d618f5bac120a2ad02/a6d36/login-new.png)

Arduino App Lab 将自动打开，并引导您完成一次性开发板配置过程，包括以下步骤：

- 设置您的**键盘**配置和**开发板名称**：

![键盘布局和开发板名称](https://docs.arduino.cc/static/4a2d80a21dd2187785273f76f8ba553a/a6d36/keyboard.png)

- 选择您的 Wi-Fi® 网络 `SSID` 并输入密码进行连接：

![Wi-Fi® 配置](https://docs.arduino.cc/static/1ce1a1578efe311b875cadd2cb3c942e/a6d36/network-new.png)

- 设置您的 Linux 登录凭据。您可以使用登录时设置的相同密码，或输入新密码进行更改：

![Linux 凭据](https://docs.arduino.cc/static/b2791d8b19724595bf9fedde418efc41/a6d36/login-credentials.png)

最终，Arduino App Lab 将打开并显示内置示例 Apps。现在，您已准备好将 UNO Q 作为单板计算机使用。

![SBC 准备就绪](https://docs.arduino.cc/static/e81adc5b20574c8b0ec83fb6567dddff/a6d36/ready-to-use.png)

> 📌 **提示：** Arduino App Lab 将始终在启动时运行，并将开发板和依赖项更新到最新版本。

### 连接到互联网

如果您按照 Arduino App Lab 首次设置操作，应该已经连接到互联网。不过，这里简要说明如何手动操作。

要将 UNO Q 连接到互联网，只需转到右上角并点击网络图标。然后，搜索可用的 Wi-Fi® 网络并选择一个。

![连接到 Wi-Fi 网络](https://docs.arduino.cc/static/1ffb5011dc268a080a7f4085920a9ef1/a6d36/wifi-new.png)

或在终端中运行以下命令：

```bash
sudo nmcli d wifi connect <SSID> password <YOUR_PASSWORD>
```

## 探索 SBC 模式：Linux 与 Arduino App Lab

现在您的单板计算机已配置完毕并准备就绪，您可以充分利用完整的 Linux 体验，并使用我们全新的 Arduino App Lab 来操作 UNO Q。

### Arduino App Lab

使用内置的 Arduino App Lab 轻松开发混合 Arduino 边缘 AI 项目。请遵循我们的[分步教程](https://docs.arduino.cc/software/app-lab/tutorials/getting-started/)来释放 Arduino UNO Q 的全部潜力。

![Arduino App Lab](https://docs.arduino.cc/static/d45264ec4c7e4a5ce4a26c5a82141905/a6d36/app-lab.png)

### 日常使用

您可以将单板计算机用于网页浏览、媒体播放、编程、文件管理等。

![桌面演示](https://docs.arduino.cc/e78b59b3037d2aed8db5ce00c4d49b30/desktop.gif)

开箱即用，UNO Q 上基于 Debian 的镜像已经预装了几个实用的应用程序，让您可以立即开始使用：

- **Chromium 浏览器：** 浏览网页、查看邮件、使用云端生产力工具。
- **终端：** 管理文件、运行 Linux 命令、安装软件、更深层次地与系统交互。
- **Vim 编辑器：** 终端中可用的强大文本编辑器，适用于编辑配置文件或编写代码。
- **文件管理器：** 在图形界面中整理您的文档、图片和下载内容。
- **媒体播放器：** 本地播放音频和视频文件（需要安装 VLC）。

这些内置应用程序为您提供了日常计算任务所需的一切，无需额外安装。当然，您始终可以使用 Debian 的软件包管理器（apt）安装更多软件来扩展您的环境。

例如，要安装 **VLC 媒体播放器**，在终端中运行以下命令：

```bash
sudo apt install vlc
```

要使用 USB 摄像头，我们可以安装 **Cheese**，这是一个提供图形界面来拍照和录像的应用程序：

```bash
sudo apt install cheese
```

![Cheese 摄像头应用](https://docs.arduino.cc/static/e74b49ef3354e587e99811d8e28f2867/a6d36/webcam.png)

## 扩展您的单板计算机

您可以通过连接额外的外设到 USB-C 转接器来为 UNO Q 单板计算机添加更多功能，例如：

- USB 摄像头
- USB 驱动器
- MicroSD 卡
- 通过以太网的有线网络连接
- 耳机（扬声器和/或麦克风）

## 网络模式

即使您已将 Arduino UNO Q 设置为**单板计算机**，您仍然可以使用 Arduino App Lab 桌面版和**网络模式**从个人电脑远程访问它。

这提供了一种灵活的方式来在电脑上开发 App，并在开发板上本地测试，因为两种模式可以同时使用。

- 当您打开 Arduino App Lab 时，您将看到开发板以 "Network" 标签列出。
- 点击它并输入 Linux 密码登录。
- 您现在可以远程访问开发板。

![网络模式](https://docs.arduino.cc/4bd91b938d5f092636a90406db84e42d/network-mode.gif)

通过这种方法，您可以从局域网内的任何机器访问您的 UNO Q。这允许您像直接连接到开发板一样使用 Arduino App Lab。开发应用程序等。

## 总结

在本教程中，您学习了如何将 Arduino UNO Q 设置并用作功能完整的单板计算机，利用其四核处理器、集成 GPU、无线连接和基于 Debian 的操作系统。您还了解了如何将硬件组装成完整的桌面系统、执行首次启动、登录和连接到互联网。

得益于其混合架构和内置的 Arduino App Lab，UNO Q 提供了通用计算和嵌入式开发的无缝融合，使您能够在无需额外硬件的情况下浏览网页、处理文档、播放媒体，并同时设计 Arduino 项目。这使其成为紧凑外形中兼具生产力和原型开发的通用平台。
