# Arduino UNO Q 4GB 开发版介绍

## 一、基本外观

Arduino UNO Q是一块小巧漂亮的开发版，集成了高通的MPU和STM32的MCU，下面来看看这块开发版的结构和功能：

![UNO 外形规格](https://docs.arduino.cc/static/7d5a122fd16435d60983ecb96c2e490f/a6d36/uno-form-factor.png)

### 板载 LED 矩阵

这块板子的一个特点是板载 **8×13 蓝色 LED 矩阵**，支持 **8 级灰度**，可实现渐变效果,由 STM32 微控制器管理。看着挺规整，不过这么多的区域放LED阵列感觉会有些冗余，如果换成板载LCD应该会更舒服。

![LED 矩阵](https://docs.arduino.cc/static/66bc34d048f9376836e3520a1ad2553d/a6d36/matrix.png)

### 四个 RGB LED

UNO Q 配备 **4 个 RGB LED**，可以显示开发板状态或通过程序控制：
- **LED 1 & 2**：由高通 MPU控制
- **LED 3 & 4**：由 STM32 MCU控制

![UNO Q RGB LED](https://docs.arduino.cc/static/10826219b80fd6d653c4601862558c1e/a6d36/rgb-led.png)

发现这里使用不同的处理器控制不同的外设，体现了双处理器架构的特点。

### 电源按钮

开发板配备物理电源按钮，**长按 5 秒以上**可重启 Linux 系统。

![UNO Q 电源按钮](https://docs.arduino.cc/static/992b8761457e587a6843b49ba732d1b4/a6d36/power-button.png)

## 二、核心构成：基于双核架构的设计

### 核心架构

Arduino UNO Q 的最大创新在于其 **混合架构**, 将高性能微处理器与实时微控制器集成在同一块板卡上。

![UNO Q 主要组件](https://docs.arduino.cc/static/567189fab6cc1a00404d38e37b42e755/a6d36/uno-q-architecture-3.png)

查询了官方文档后，我来简要总结一下处理器的各种参数：

### MPU：Qualcomm QRB2210

| 规格 | 参数 |
|------|------|
| 核心 | 四核 Arm Cortex-A53 |
| 频率 | 2.0 GHz |
| GPU | Adreno 702 @ 845 MHz |
| ISP | 双 ISP，支持 25 MP @ 30 fps |
| 操作系统 | Debian Linux（上游支持） |

这款处理器专为嵌入式视觉和边缘计算应用设计，具备强大的 AI 推理能力。

### MCU：STM32U585

| 规格 | 参数 |
|------|------|
| 核心 | Arm Cortex-M33 |
| 频率 | 高达 160 MHz |
| Flash | 2 MB |
| SRAM | 786 KB |
| 操作系统 | Zephyr OS |

运行 Arduino sketches，为低功耗嵌入式应用提供安全高效的平台。

---
我非常喜欢这块开发版的混合架构，其为上位机和下位机的协同工作提供了优雅的解决方案。Arduino打通了MPU和MCU之间的沟通链，在编写代码时只需使用Bridge函数即可实现MPU和MCU的高效通信。比如如果想做一个视觉云台的话，只需这一块板子就可以实现视觉处理和云台控制了。
### 无线连接模块

WCBN3536A 无线模块提供：
- 双频 Wi-Fi 5（2.4/5 GHz）
- Bluetooth 5.1
- 板载天线

### 存储配置

| 配置选项 | 规格 |
|----------|------|
| eMMC 存储 | 32 GB |
| LPDDR4 RAM | 4 GB |


### 其他关键组件

- **多媒体编解码器**：ANX7625，支持视频/音频输出
- **电源管理**：Qualcomm PM4145 PMIC

## 三、外设接口

### 引脚布局

UNO Q 配备两种类型的连接器：
1. **顶部**：经典 UNO 风格接头，用于原型设计
2. **底部**：高速接头连接器，用于载板集成

![UNO Q 简易引脚图](https://docs.arduino.cc/static/c4c115ced208022ab43299bda7ea661e/a6d36/Simple-pinout-ABX00162.png)

### 数字引脚

UNO Q 有 **47 个数字引脚**由 STM32 微控制器控制：
- 22 个通过 UNO 风格连接器引出
- 25 个通过 JMISC 连接器引出

### 模拟引脚

**6 个 14 位 ADC 引脚**（JANALOG 连接器）：

| 引脚 | 功能 |
|------|------|
| A0/A1 | ADC / DAC |
| A2/A3 | ADC / OPAMP |
| A4/A5 | ADC / I2C |

ADC 分辨率可配置为 14/12/10/8 位，电压参考支持多种内部/外部选项。

### USB-C 连接器

这是 UNO Q 的核心接口：

![USB-C 连接器](https://docs.arduino.cc/static/75abc75f671a776ae751e30a4080fc51/a6d36/usb.png)

| 特性 | 规格 |
|------|------|
| USB 电源 | 5 VDC 3 A (15 W) |
| USB 标准 | USB 3.1 Gen 1 (5 Gb/s) |
| 视频输出 | DisplayPort |

通过 USB-C dongle 可扩展：
- **视频**：HDMI 输出、USB 摄像头输入
- **音频**：USB 或 3.5mm 耳机
- **网络**：以太网
- **存储**：microSD 卡、USB 驱动器

---
这块板子只有1个USB接口，就说明如果要实现数据传输和供电的功能的话，就只能使用拓展坞或者使用网络连接电脑了。如果能设置更多的接口也许会更好。

### 通信接口

| 接口 | 用途 |
|------|------|
| **SPI** | 高速串行通信 |
| **I2C** | 多设备总线通信 |
| **Qwiic** | I2C 生态系统 |
| **UART** | 串行通信 |
| **CAN 总线** | 工业级通信 |

### 调试接口

**JCTL 连接器**提供专用调试 UART：
- 波特率：115200 bps
- 逻辑电平：**1.8V**（需注意电压匹配!）

![调试 shell](https://docs.arduino.cc/40e41d2593a4441e3ceb16d3be558d40/debug-shell.gif)

### 双处理器协同

Bridge 工具实现了 MPU 与 MCU 之间的高效通信：

```python
# 从 Sketch 发送数据到 Python
Bridge.notify("python_function", data);

# Python 端接收
Bridge.provide("python_function", python_function)
```

## 六、供电系统

这里需要特别注意，由于这块板子搭载了2块不同的处理器，所以它的供电系统比较特殊，外设引脚的耐受电压也有所差异，我在网上已经看到一些由于供电不当或是错误输入引脚电压导致烧坏板子的案例了，所以在第一次给板子供电之前要先仔细阅读系统供电的要求和限制！

---

### 三种供电方式

![UNO Q 供电选项](https://docs.arduino.cc/static/72456f6873252eb705cfd28538166e8a/a6d36/power-options-3.png)

| 供电方式 | 电压 | 电流要求 | 连接位置 |
|----------|------|----------|----------|
| **USB-C** | 5V DC | **≥ 3A** | USB-C 接口 |
| **5V 引脚** | 5V DC | ≥ 3A | JANALOG (A3) |
| **VIN 引脚** | 7-24V DC | 根据负载 | JMEDIA (B2) |

---
日常使用的时候用USB-C供电就行了，要注意供电最低要求是5V3A，线也最好用好一点的,需要支持数据传输。比如我将板子和电脑连接，但是电脑A口最大输出电流仅0.9A，开发板可能因为供电不足反复重启。所以我使用双Type-C数据线连接开发板和电脑C口，开发板即可正常运行。

### 1.7 连接器电压要求

![UNO Q Headers](https://docs.arduino.cc/static/b2fce2381981005a2e8b7c8fbe2fd0c9/a6d36/uno-q-headers.png)

简单来说就是背面的接口是高通MPU引出的，大多只支持1.8V电压输入，正面的电压要求也不尽相同，使用时要注意一下。

---

注意模拟输入的电压要求：
![UNO Q Analog Inputs](https://docs.arduino.cc/static/84e6734b392959199802697eea8b32fe/a6d36/uno-q-analog.png)

- 参考电压 VREF+ ≈ 3.3V
- **不支持 5V 输入**
- 绝对最大值：VDD + 0.3V ≈ 3.6V
