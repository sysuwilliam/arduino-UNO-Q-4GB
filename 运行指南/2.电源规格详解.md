## 一、概述

本教程详细讲解 UNO Q 的电源域，重点关注三个主要电压轨：**5V**、**3.3V** 和 **1.8V**。你将学习：

- 5V 输入如何被接受并进行二极管 OR 组合
- 如何转换为 3.8V 预稳压节点
- 如何稳压为 3.3V
- PMIC 如何从 5V 输入生成 1.8V 轨
- 转换、保护和路由这些轨的组件
- 安全电压和引脚级限制

## 四、电源规格

### 4.1 输入电源

| 电源来源 | 电压范围 | 推荐电流 | 连接器 |
|----------|----------|----------|--------|
| **USB-C VBUS** | 5V | ≥ 3A | USB-C 连接器 |
| **VIN (DC IN)** | 7-24V | 根据 5V 预算配置 | JMEDIA, JANALOG |
| **5V 引脚** | 5V | ≥ 3A | JANALOG |

#### 重要说明

> **⚠️ 重要：** 使用稳定的 5V/3A 电源。如果电源在峰值时电流限制，电压下降可能导致复位。

### 4.2 USB-C 能力

| 特性 | 说明 |
|------|------|
| **USB 标准** | USB 3.1，支持角色切换 |
| **DisplayPort** | DisplayPort Alt-Mode 视频（通过 ANX7625 DSI-to-DP 桥接） |
| **USB PD** | 仅请求 **5V/3A** 合同，不请求更高电压配置 |
| **注意** | DisplayPort Alt-Mode 激活时，USB 数据速度降低 |

### 4.3 推荐工作条件

| 参数 | 符号 | 最小值 | 典型值 | 最大值 | 单位 |
|------|------|--------|--------|--------|------|
| USB-C 输入 | VBUS_USBC | 4.5 | 5.0 | 5.5 | V |
| DC 输入 (VIN) | DC_IN | 7.0 | - | 24.0 | V |
| 3.3V 系统轨 | PWR_3P3V | 3.1 | 3.3 | 3.5 | V |
| 工作温度 | T_OP | -10 | - | 60 | °C |

#### 参数说明

| 术语 | 定义 |
|------|------|
| **最小值** | 正常运行所需的最低连续值，短暂下降可能导致复位或链路断开 |
| **典型值** | 标称设计点，用于规划 |
| **最大值** | 不得超出此值 |

## 五、板载电压轨详解

### 5.1 电压轨总览

| 电压 | 轨名称 | 来源/稳压器 |
|------|--------|-------------|
| **5.0V** | `5V_SYS` | USB-C VBUS 与 7-24V 降压输出的二极管 OR 组合 |
| **3.8V** | `PWR_3P8V` | TPS62A02A 降压（从 5V_SYS） |
| **3.3V** | `PWR_3P3V` | TPS62A02A 降压（从 PWR_3P8V） |
| **1.8V** | `VREG_L15A_1P8V` | PM4125 LDO L15A（从 5V_SYS） |

### 5.2 5V 轨 (5V_SYS)

**`5V_SYS`** 是板子的系统 5V 总线。

#### 功能

- 为两个降压转换器供电（转换到 3.8V 和 3.3V）
- 当板子作为 Host/OTG 时，提供 USB VBUS 反向驱动路径
- 通过 P-channel MOSFET 切换反向驱动路径

#### 连接器上的表示

在连接器上表示为 `VBUS_USBC`，是一个 5V 总线。

### 5.3 3.8V 轨 (PWR_3P8V)

**`PWR_3P8V`** 是中间轨，由 TPS62A02APDDCR 降压转换器从 5V_SYS 产生。

#### 功能

- 为 3.3V 降压转换器供电
- 连接到 `VBAT`（用于系统设计和未来功能）

> ⚠️ **注意：** 此电压轨保留用于系统设计和未来功能。

### 5.4 3.3V 轨 (PWR_3P3V)

**`PWR_3P3V`** 是板子的主要轨，用于通用逻辑和外设。

#### 供电对象

| 组件 | 电源域 |
|------|--------|
| **STM32U585** | VDD, VDDA, VDDUSB, VREF+ |
| **ANX7625** | AVDD33, VDDIO_3V3 |
| **电平转换器** | VCCB（3.3V 侧） |
| **板上 3.3V 设备** | - |
| **导出引脚** | JANALOG, JDIGITAL, JMISC, QWIIC |

#### PMIC 内部 3.3V 轨

PMIC 内部有额外的 3.3V LDO 轨，例如 `VREG_L22A_3P3V` 用于射频/PA 偏置。

### 5.5 1.8V 轨 (VREG_L15A_1P8V)

**`VREG_L15A_1P8V`** 是 PMIC 输出的导出 1.8V 轨。

#### 供电对象

| 组件 | 电源域 |
|------|--------|
| **QRB2210** | I/O banks 和高速接口 |
| **ANX7625** | DVDD18 域 |
| **电平转换器** | VCCA（1.8V 侧） |
| **Wi-Fi/蓝牙** | 数字 I/O |
| **导出引脚** | JMISC, JCTL 的 1.8V 引脚 |

> ⚠️ **重要：** 不要将 3.3V 信号接入此电压域，以避免处理器级 I/O 过压。

## 六、PMIC PM4125 域

### 6.1 PMIC 定义

**PMIC（Power Management IC）** 是电源管理集成电路，用于：

- 从上游电源创建和控制多个电源轨
- 提供降压转换和低压差（LDO）线性稳压
- 管理轨使能和时序
- 状态报告
- 电池充电和保护

### 6.2 PM4125 功能

UNO Q 使用 **PM4125** 作为 PMIC。

#### 输入与输出

```
┌────────────────────────────────────────────────────────────────┐
│                      PMIC PM4125                                │
├────────────────────────────────────────────────────────────────┤
│                                                                 │
│   输入: 5V_SYS                                                  │
│                                                                 │
│   ├── 内部处理器/内存轨                                         │
│   │   ├── VDD2 (~1.1V)        - 核心电源                       │
│   │   ├── VDDQ (~0.6V)        - DDR 电源                       │
│   │   ├── VPP (1.8V)          - DDR VPP                        │
│   │   └── VREF_DDR (~0.6V)    - DDR 参考                       │
│   │                                                            │
│   ├── 导出轨                                                    │
│   │   └── VREG_L15A_1P8V (1.8V) - 板载 1.8V 轨                 │
│   │                                                            │
│   ├── eMMC 电源                                                 │
│   │   ├── VCC (3.3V 或 1.8V)   - 根据配置                      │
│   │   └── VCCQ (1.8V)                                         │
│   │                                                            │
│   └── 内部 3.3V LDO                                             │
│       └── VREG_L22A_3P3V       - 射频/PA 偏置                  │
│                                                                 │
└────────────────────────────────────────────────────────────────┘
```

#### 充电模块监控

| 信号 | 说明 |
|------|------|
| `USB_VBUS_IN` | VBUS 存在时监控 |
| `VSW_CHG` | 充电路径 |
| `VPH_PWR` | 电源路径 |
| `VBAT` | 电池感测 |

## 七、关键功率组件

### 7.1 组件列表

| 参考编号 | 组件 | 功能 |
|----------|------|------|
| D2801, D2803 | SX34 Schottky 二极管 | 二极管 OR 组合 |
| U2803 | LMR51440 降压转换器 | 7-24V → 5V |
| U2801 | TPS62A02A 降压转换器 | 5V → 3.8V |
| U2802 | TPS62A02A 降压转换器 | 3.8V → 3.3V |
| U3004 | TPS7A2030 LDO 稳压器 | 3.3V → 3.0V |
| Q2801 | PJA3413 P-MOSFET | VBUS 开关 |
| Q2802 | MMBT3904 NPN 晶体管 | 使能门控 |
| PMIC1 | PM-4125-3-NSP194-TR-01-0 | 电源管理 IC |
| U3001 | ANX7625 | DSI 转 DisplayPort 桥接 |
| MCU1 | STM32U585 | 微控制器 |

### 7.2 组件功能详解

#### Schottky 整流器 (D2801, D2803)

- **功能：** 提供二极管 OR，将 USB-C VBUS 路径和 DC 降压的 5V 输出合并到系统总线 `5V_SYS`

#### LMR51440 降压转换器 (U2803)

- **功能：** 宽输入转换器，将 7-24V 降压到 5V
- **输出：** `5V_BUCK_OUT`，通过 `D2803` 进入 OR 级

#### TPS62A02A 降压转换器 (U2801)

- **功能：** 从 `5V_SYS` 输出 3.8V 预稳压节点 `PWR_3P8V`
- **说明：** 此节点为 3.3V 转换器供电

#### TPS62A02A 降压转换器 (U2802)

- **功能：** 从 `PWR_3P8V` 输出 3.3V 轨 `PWR_3P3V`
- **供电对象：**
  - STM32U585 的 VDD, VDDA, VDDUSB, VREF+
  - 电平转换器 3.3V 侧 (VCCB)
  - 板上 3.3V 设备
  - 导出的 3.3V 引脚

#### TPS7A2030 LDO 稳压器 (U3004)

- **功能：** 从 `PWR_3P3V` 输出 3.0V 模拟轨
- **用途：** ANX7625 的 `AVD03P0V`

#### P-MOSFET (Q2801)

- **功能：** 将 `5V_SYS` 切换到 `USB_VBUS_IN`，用于 Host 或 OTG 反向驱动
- **使能：** `USB_VBUS_ON_N`（低电平有效）
- **门控：** 通过 Q2802 和 `LDO1_PG` 控制

#### PMIC PM4125

- **功能：** 从 `5V_SYS` 生成内部处理器和内存轨
- **导出：** `VREG_L15A_1P8V` 作为板载 1.8V 轨

#### ANX7625 (DSI 转 DisplayPort 桥接)

| 电源轨 | 来源 |
|--------|------|
| VDDIO_3V3 | PWR_3P3V |
| DVDD18 | VREG_L15A_1P8V |
| AVD03P0V | U3004 (3.0V LDO) |

#### STM32U585 微控制器

| 引脚 | 电源来源 |
|------|----------|
| VDD, VDDA, VDDUSB, VREF+ | PWR_3P3V |
| VDDIO2 (1.8V bank) | VREG_L15A_1P8V（通过电平转换器） |

## 八、引脚级电源容差

### 8.1 I/O 银行电压域

| 信号银行 | 标称 I/O | 绝对最大值 |
|----------|----------|------------|
| **处理器 I/O** - JMEDIA, JMISC, JCTL | 1.8V | 2.1V |
| **Maker I/O** - JDIGITAL, JANALOG | 3.3V | 3.6V |

### 8.2 处理器 I/O（1.8V 域）

- **连接位置：** JMEDIA, JMISC, JCTL
- **直接连接：** Qualcomm Dragonwing™ QRB2210 引脚
- **电压限制：** 仅使用 1.8V 逻辑
- **绝对最大值：** 2.1V

> ⚠️ **警告：** 不要用 3.3V 信号驱动 1.8V 银行！

### 8.3 Maker I/O（3.3V 域）

- **连接位置：** JDIGITAL, JANALOG
- **参考电源：** `PWR_3P3V`
- **电平转换：** 进入处理器 1.8V 域时经过电平转换
- **绝对最大值：** 3.6V

### 8.4 I/O 操作电压和驱动能力

#### JDIGITAL（MCU GPIO @ 3.3V）

- 遵循 STM32U585 电气规格
- 根据电气限制规划负载
- 尊重每个端口的总电流

#### JANALOG（A0/A1）

- **仅 ADC 输入**
- **不要**向这些引脚输入或输出 DC 电流
- 更高电压感测时使用电阻分压器

#### JMEDIA（MIPI DSI/CSI @ 1.8V）

- 高速 MIPI DSI/CSI 通道
- **不是**通用 I/O
- **不要**添加上拉、下拉或其他 DC 负载
- 载板上保持受控阻抗

## 九、连接器电源轨

### 9.1 连接器电源分布

| 连接器 | 导出电源轨 |
|--------|------------|
| **JCTL (A1)** | +1.8V (OUT), GND |
| **JDIGITAL (A2)** | PWR_3P3V (3.3V) |
| **JANALOG (A3)** | PWR_3P3V (3.3V), 5V_USB_VBUS (5V) |
| **QWIIC (A4)** | PWR_3P3V (3.3V) |
| **JSPI (A5)** | 5V_USB_VBUS (5V) |
| **JMISC (B1)** | PWR_3P3V (3.3V), VREG_L15A_1P8V (1.8V), 5V_USB_VBUS (5V), VBAT (3.8V) |
| **JMEDIA (B2)** | PWR_3P3V (3.3V), VIN/DC_IN (7-24V) |

### 9.2 详细说明

#### JCTL (A1)

- **逻辑电平：** 1.8V 域
- **电源输出：** +1.8V 参考 (OUT), GND
- **警告：** 不要用 3.3V 驱动这些引脚

#### JDIGITAL (A2)

- **逻辑电平：** 3.3V
- **电源输出：** PWR_3P3V (3.3V)

#### JANALOG (A3)

- **逻辑电平：** 3.3V
- **电源输出：**
  - PWR_3P3V (3.3V)
  - 5V_USB_VBUS (5V) - 仅电源

#### QWIIC (A4)

- **逻辑电平：** 3.3V
- **电源输出：** PWR_3P3V (3.3V)
- **用途：** I2C Qwiic 生态系统

#### JSPI (A5)

- **逻辑电平：** 3.3V（SPI 信号）
- **电源输出：** 5V_USB_VBUS (5V) - 用于需要 5V 的配件
- **特性：** STM32U585 SPI 引脚作为输入时支持 5V 容忍，输出驱动 3.3V

#### JMISC (B1)

- **混合电压：**
  - 3.3V MCU 信号
  - 1.8V MPU 信号
  - 模拟音频
- **电源输出：**
  - PWR_3P3V (3.3V)
  - VREG_L15A_1P8V (1.8V)
  - 5V_USB_VBUS (5V)
  - VBAT (3.8V) - 保留

#### JMEDIA (B2)

- **逻辑电平：** 1.8V MIPI DSI/CSI
- **电源输出：**
  - PWR_3P3V (3.3V) - 用于 3.3V 外设
  - VIN/DC_IN (7-24V) - 原始输入，**不是**逻辑轨

## 十、电流限制

### 10.1 电源共享原则

| 说明 |
|------|
| 连接器电源与板上负载共享 |
| `5V_USB_VBUS` 和 `PWR_3P3V` 的可用电流取决于输入源、稳压器和板子自身消耗 |
| 在相同轨预算内规划外设，并在预期负载和温度下验证性能 |

### 10.2 接地建议

| 建议 |
|------|
| 从连接器供电时，使用附近的 GND 引脚作为返回路径 |
| 更高负载时，使用多个 GND 返回以减少电压降和噪声 |

## 十一、USB-C VBUS 反向驱动

### 11.1 工作原理

P-channel MOSFET (Q2801) 将 5V 系统总线 (`5V_SYS`) 连接到 `USB_VBUS_IN`。

```
┌─────────────────────────────────────────────────────────────────┐
│                    VBUS 反向驱动控制                             │
├─────────────────────────────────────────────────────────────────┤
│                                                                  │
│   5V_SYS ──┬── Q2801 (P-MOSFET) ──► USB_VBUS_IN                 │
│             │                                                   │
│             │    使能控制                                        │
│             │    ┌─────┐                                        │
│             └────┤ Q2802├── LDO1_PG                             │
│                  │ NPN │                                        │
│                  └──┬──┘                                         │
│                     │                                            │
│                     ▼                                            │
│              USB_VBUS_ON_N (低电平有效)                          │
│                                                                  │
│   条件：轨有效时才驱动 VBUS                                      │
│                                                                  │
└─────────────────────────────────────────────────────────────────┘
```

### 11.2 使能控制

| 组件 | 功能 |
|------|------|
| Q2801 (P-MOSFET) | 开关 VBUS |
| Q2802 (NPN) | 门控使能 |
| USB_VBUS_ON_N | 低电平有效使能信号 |
| LDO1_PG | 电源良好信号 |

### 11.3 使用场景

| 场景 | VBUS 状态 |
|------|-----------|
| **USB Host** | 板子提供 5V VBUS 给外设 |
| **OTG Host** | 板子提供 5V VBUS |
| **USB Device** | 不驱动 VBUS |

## 十二、模拟输入

### 12.1 A0/A1 特性

| 参数 | 说明 |
|------|------|
| **引脚** | PA4 (A0), PA5 (A1) |
| **连接** | 直接 STM32U585 ADC 输入 |
| **参考** | VREF+ (3.3V 轨) |
| **5V 容忍** | ❌ **不支持 5V** |
| **有效范围** | 0V ~ VREF+ |
| **绝对最大** | VDD + 0.3V ≈ **3.6V** |

### 12.2 保护机制

超过 VDD + 0.3V 时：
- 片上保护网络钳位
- 注入电流流动
- 可能损坏设备或破坏测量

### 12.3 高压测量方法

#### 电阻分压器法

```
测量 5V 信号的分压器电路：

5V 信号 ──┬── 10kΩ ──┬──► ADC 引脚 (约 3.3V)
          │          │
          │        100nF (去耦)
          │          │
          └── 20kΩ ──┴── GND

计算：5V × (20kΩ / (10kΩ + 20kΩ)) ≈ 3.3V
```

#### 建议配置

| 元件 | 值 | 功能 |
|------|-----|------|
| 上电阻 | 10kΩ | 分压 |
| 下电阻 | 20kΩ | 分压 |
| 电容 | 100nF | 抗混叠、降噪 |
| 串联电阻 | ~1kΩ | 限制注入电流 |

#### 注意事项

- 保持有效源阻抗 ≤ 50kΩ，或增加 ADC 采样时间
- A4/A5 作为 I2C3 (PC1/PC0) 时，仅使用 3.3V 上拉

### 12.4 其他模拟引脚

| 引脚 | MCU 引脚 | 5V 容忍 |
|------|----------|---------|
| A2 | PA6 | 数字模式支持，ADC 模式不支持 |
| A3 | PA7 | 数字模式支持，ADC 模式不支持 |
| A4 | PC1 | 数字模式支持，ADC 模式不支持 |
| A5 | PC0 | 数字模式支持，ADC 模式不支持 |

> ⚠️ 所有模拟功能配置下，必须保持在 VDD + 0.3V 以内！

| 顺序 | 动作 |
|------|------|
| 1 | 1.8V 轨关闭 |

### 14.3 时序目的

## 十六、总结

### 16.1 电源架构要点

| 要点 | 说明 |
|------|------|
| **输入** | USB-C 5V 或 VIN 7-24V |
| **合并** | 二极管 OR 组合到 5V_SYS |
| **中间级** | 3.8V 预稳压 (PWR_3P8V) |
| **主轨** | 3.3V (PWR_3P3V) 和 1.8V (VREG_L15A_1P8V) |
| **PMIC** | PM4125 从 5V_SYS 生成内部轨 |

### 16.2 关键限制

| 参数 | 限制 |
|------|------|
| **USB-C 输入** | 5V/3A |
| **VIN 输入** | 7-24V |
| **处理器 I/O** | 1.8V 标称，最大 2.1V |
| **Maker I/O** | 3.3V 标称，最大 3.6V |
| **模拟输入 A0/A1** | 最大 VDD + 0.3V (≈3.6V)，**不支持 5V** |

### 16.3 应用指南

| 任务 | 方法 |
|------|------|
| **选择电源** | 使用稳定的 5V/3A 电源 |
| **预算负载** | 在相同轨预算内规划外设 |
| **验证启动** | 检查轨顺序和连接器轨 |
| **跨域连接** | 使用电平转换器 |
