## 发布版本

### Release 0.6.0 [2026.03.18]

通过新的开发板设置页面，刷写和更新开发板将变得更加容易！设置您的开发板，并在此最新版本中探索新的 Bricks 和示例。

新示例：
- Telegram Bot
- Music Composer
- Cloud AI Assistant

请参阅下面的完整新增功能和修复错误列表。

#### 新增功能

- 新的开发板设置页面，具有更新和刷写功能
- 当模型重新训练时的 Edge Impulse 模型更新操作
- UI 更新
  - 设备发现页面
  - Edge Impulse 集成
- Bricks
  - 新增：Sound Generator
  - 新增：Telegram Bot
  - 新增：Automatic Speech Recognition (Cloud)
  - 改进：Wave Generator
  - 所有 Bricks 现在都有基本使用示例
- 示例
  - 新增：Telegram Bot
  - 新增：Music Composer
  - 新增：Cloud AI Assistant
  - 改进："Led Matrix Painter" sketch 的线程安全性

#### 修复内容

- 编辑器文件树和 app 编辑操作问题
- 无障碍改进（更好的键盘支持覆盖）
- 示例所需的 Sketch 库现在作为软件更新的一部分下载
- Arduino Apps 描述现在从 readme.md 文件获取（如果 app.yaml 文件中不存在）
- arduino-router 包现在提供 CLI（arduino-router-client）来手动发送消息，用于测试和调试目的
- 当 sketch 未更改时改进编译速度
- Bricks
  - 修复音频外设枚举
  - 在错误情况下记录 LLM 提供商日志
- 示例
  - 修复 USB 模式下的"Detect objects on smartphone camera"示例
  - 修复多重检测情况下的"Detect objects on smartphone camera"示例
  - 更新 sketch 中使用的库

您始终可以在[这里](https://github.com/arduino/arduino-app-lab/releases)找到最新版本。

## 历史版本

以下是以前版本的发布说明。

### Release 0.5.0 [2026.02.27]

**详情**

您现在可以将 App Lab 与 Arduino Cloud 和您的 Edge Impulse 账户连接！这意味着您现在可以使用 IoT Remote App 作为示例的摄像头输入，以及使用 Edge Impulse 创建自己的自定义模型。

新示例：
- Color Your LEDs
- Video Object Detection on Mobile

请参阅下面的完整新增功能和修复错误列表。

#### 新增功能

- 将 App Lab 与 Edge Impulse 账户连接
- Edge Impulse 自定义模型集成
- 从页脚切换开发板
- 使用命令行导入/导出 Arduino Apps
- UI 更新
  - 更新期间的新功能弹出窗口
  - 通知更新
  - 选项卡更改或重启时保持编辑器面板大小
  - 其他 UI 改进
- Bricks
  - 新增：Web Socket Camera 与 Arduino IoT Remote App 集成
  - 改进：Camera peripheral、brick 文档
- 示例
  - 新增：Color your LEDs
  - 新增：Video Object Detection on Mobile
  - 改进：LED Matrix Painter

#### 修复内容

- 滚动时 HTML 语法高亮丢失
- 开发板发现：当 USB 连接可用时显示重复项和网络模式
- 当"check for updates"失败时允许跳过/更改网络
- 其他外观/行为缺陷
- 更新现在只下载最小必需数据，而不是每次重新下载所有容器镜像
- app-cli 命令现在也会自动补全 app 名称为"folder names"
- 修复检查更新但没有平台更新时的错误
- Bricks
  - Camera peripheral discovery 和 Speaker peripheral discovery 的次要修复
- 示例
  - 多重检测情况下的 Object Detection 修复

#### GitHub 发布

[https://github.com/arduino/arduino-app-lab/releases/tag/al-0.5.0](https://github.com/arduino/arduino-app-lab/releases/tag/al-0.5.0)

### Release 0.4.0 [2026.02.06]

**详情**

#### 新增功能

- Flasher 工具集成到 App Lab（用于过期的开发板）
- 跳过 WiFi 步骤并显示警告（WiFi 相关功能尚未管理）
- Web 开发代码的语法高亮（js/ts、css/scss、html、json）
- 分析增强以区分 SBC/PC 用户
- 当 Arduino Apps 意外崩溃时保留 app 日志
- 编辑器 QOL 更新
- UI 重构

#### 修复内容

- app/brick 配置时 App.yaml/README.md 消失
- Arduino App UI 偶尔无法打开
- 连接网络模式时未处理"wrong password"
- SBC/网络模式下 UI 中损坏的 README.md 加载
- 开源仓库的安全修复
- 其他行为和外观修正

#### GitHub 发布

[https://github.com/arduino/arduino-app-lab/releases/tag/al-0.4.0](https://github.com/arduino/arduino-app-lab/releases/tag/al-0.4.0)

如果您已经下载过 App Lab，下次打开软件时应该会自动获取更新。您也可以[在这里下载最新版本](https://www.arduino.cc/en/software/#app-lab-section)。
